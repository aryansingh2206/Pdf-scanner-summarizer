# 📄 PDF Scanner & Summarizer 🚀  

Turn lengthy PDFs into bite-sized insights with AI-powered summarization!  

## ✨ Features  
✅ **Upload PDFs** – Drop in your files and let AI do the magic.  
✅ **AI-Powered Summarization** – Get concise, readable summaries in seconds.  
✅ **User-Friendly Interface** – No tech knowledge required, just upload & read!  
✅ **Fast & Secure** – Your data stays yours.  

## 🛠️ Tech Stack  
- **Frontend:** React.js (Styled Components for modern styling)  
- **Backend:** Node.js + Express.js  
- **AI Summarization:** NLP & Machine Learning  
- **Database:** PostgreSQL
